One time setup

Do this on each computer

If you can connect with two devices, you can skip this step. If you are connecting with a single device, it is useful to be able to have the "Brady Bunch" view of the board and all the players, and for that you (sadly) need to use the Google Chrome browser, with an add-on. Install Google's Chrome browser from here. In it, navigate to the app store, and install the "meet grid" add-on.

Playing

Ready to play?

Get the meeting invite from the host and join the meeting. You will probably want to enable the grid view, so that you can see the board and the players at the same time. Alternatively, connect with two devices and use one a larger one locked onto the board. It works nicely if you can screen cast to a large TV to see the board and other players.

Hosting

Thinking of hosting?

Since you are the one with the board, you will need one camera to point at it, and one camera to point at you. The board camera can be your phone or a webcam, so you can use your computer for the game play.
