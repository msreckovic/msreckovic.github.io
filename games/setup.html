Install and Customize Google Chrome Browser

Install Google Chrome Browser

Skip this step if you have Google Chrome already installed.

Navigate (in any web browser) to the Chrome installation page and follow the installation instructions. You do not need to make Google Chrome your default browser. After the installation is complete, reopen this page in Google Chrome.

Install The Grid Add-On

You should be following this step in Google Chrome browser.

Navigate to the Chrome app store, and follow the instructions to install the Google Meet Grid View.
